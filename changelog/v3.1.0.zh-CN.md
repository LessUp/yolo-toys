## ✨ 新特性

### 专业化文档系统

完全重构的文档体系，包含 20+ 新增和更新文档：

| 分类 | 文档 |
|------|------|
| 入门指南 | 安装指南、快速开始（中/英文） |
| API 参考 | REST API、WebSocket 协议（中/英文） |
| 架构设计 | 系统概述、处理器模式（中/英文） |
| 部署指南 | Docker 指南、环境变量参考 |
| 开发指南 | 添加自定义模型（中/英文） |
| 参考手册 | 模型列表、FAQ（中/英文） |

### 双语支持

所有文档现提供：
- 🇺🇸 **English** — 主要语言
- 🇨🇳 **简体中文** — 完整翻译

### 标准化更新日志

采纳 [Keep a Changelog](https://keepachangelog.com/) 格式：
- CHANGELOG.md — 英文版本
- CHANGELOG.zh-CN.md — 中文版本
- 版本发布说明（v3.1.0.md、v3.0.0.md 等）
- 历史归档文件保留
