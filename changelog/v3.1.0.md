# YOLO-Toys v3.1.0 Release Notes

**Release Date:** 2026-04-16

---

## 📝 Documentation System Refactor

This release marks a major milestone in project documentation, introducing a professional bilingual documentation system and standardized changelog format.

---

## ✨ What's New

### Professional Documentation System

Completely restructured documentation with 20+ new and updated documents:

| Category | Documents |
|----------|-----------|
| Getting Started | Installation Guide, Quick Start (EN/ZH) |
| API Reference | REST API, WebSocket Protocol (EN/ZH) |
| Architecture | System Overview, Handler Pattern (EN/ZH) |
| Deployment | Docker Guide, Environment Reference |
| Guides | Adding Custom Models (EN/ZH) |
| Reference | Models, FAQ (EN/ZH) |

### Bilingual Support

All documentation is now available in:
- 🇺🇸 **English** — Primary language
- 🇨🇳 **简体中文** — Full translation

### Standardized Changelog

Adopted [Keep a Changelog](https://keepachangelog.com/) format:
- CHANGELOG.md — English version
- CHANGELOG.zh-CN.md — Chinese version
- Version release notes (v3.1.0.md, v3.0.0.md, etc.)
- Historical archives preserved

- Update `docs/README.md` when adding new sections
